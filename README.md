# MLProjects

A collection of small machine learning experiments built with Python and Jupyter notebooks.

## Projects

| Project | Description |
|---|---|
| [Spam Mail Prediction](Spam%20Mail%20Prediction/) | Classifies emails as spam or ham using logistic regression |
| [Wine Quality](WineQuality/) | Predicts wine quality using Random Forest |
| [Diabetes Prediction](Diabetes%20Prediction/) | Binary classification to predict diabetes onset |
| [Movie Recommender System](Movie%20Recommender%20System/) | Content-based movie recommendations |
| [Credit Card Fraud](Credit%20Card%20Fraud/) | Anomaly detection for fraudulent transactions |
| [Rainfall Prediction](Rainfall%20Prediction/) | Predicts rainfall using weather features |
| [Computer Engineers Salary Prediction](Computer%20Engineers%20Salary%20Prediction/) | Salary estimation for engineering graduates |
| [YouTube Trending Videos](YouTubeTrendingVideos/) | EDA on US YouTube trending video data |

## Getting Started

### 1. Clone
```bash
git clone https://github.com/eisenheiim/MLProjects.git
cd MLProjects
```

### 2. Create a virtual environment (recommended)
```bash
python -m venv .venv
# macOS/Linux:
source .venv/bin/activate
# Windows:
.venv\Scripts\activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Launch Jupyter
```bash
jupyter notebook
```

Open any `.ipynb` file from the project folders to explore.

## Notes
- Notebooks contain inline comments explaining each step.
- Datasets are included in their respective folders where file size permits.
- Experiments are exploratory; model results may vary with different random seeds.
